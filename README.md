# Fortynotes – Landingpage

Dies ist die offizielle Landingpage für **Fortynotes**, die minimalistische, privacy‑first Notiz‑App von fortyninelabs.

Die Seite ist mit **React**, **Vite** und **Tailwind CSS** gebaut und wird über **Netlify** bereitgestellt.

## 🌐 Live‑Website

https://fortynotes.netlify.app/

## 🚀 Entwicklung

```bash
npm install
npm run dev
```
